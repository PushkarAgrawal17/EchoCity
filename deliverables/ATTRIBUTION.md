# EchoCity — Attributions & Open-Source Credits

This document lists the open-source libraries, pretrained models, datasets, frameworks, and developer tools leveraged to build and execute the EchoCity local runtime.

---

## 1. Local AI & Model Weights

*   **SmolLM2-1.7B-Instruct**:
    *   *Creator*: Hugging Face Llama/SmolLM Team.
    *   *Description*: A state-of-the-art 1.7 Billion parameter language model optimized for local consumer CPU execution.
    *   *License*: Apache 2.0.
*   **Ollama**:
    *   *Creator*: Ollama team.
    *   *Description*: Local lightweight model execution engine built upon the `llama.cpp` runtime.
    *   *License*: MIT.
